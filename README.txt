JavaScript Calculator

Built based on the JavaScript Calculator challenge from freeCodeCamp
https://www.freecodecamp.com/challenges/build-a-javascript-calculator


Details:

Goal of the project was to practice javascript and frontend development
Four basic arithmetic operators available
- Multiplication, Addition, Subtraction, Division
Supports decimals
Operations can be chained, meaning a value from a previous calculation can be used again in your next calculation

Technologies:
HTML, CSS/Bootstrap, JavaScript/jQuery
